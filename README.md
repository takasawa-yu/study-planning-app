# 資格スタディ

記録と計画に集中する、資格試験の学習アプリ。単一HTMLファイルで動き、データはブラウザの `localStorage` にのみ保存される（サーバー送信なし）。

**アプリを開く → https://takasawa-yu.github.io/study-planning-app/**

スマホのブラウザで開いて「ホーム画面に追加」すると、アプリのように起動できる（PWA対応・オフラインでも起動可）。

## できること

- **プロジェクト（試験）と教材の管理** — 試験ごとに教材・タスクを登録。教材は複数の試験で共有できる
- **記録** — タイマー（一時停止・再開対応）または手動で、勉強時間と進んだ量（ページ／問題数／任意の単位）を記録
- **計画** — タスクの工数と1日の勉強可能時間から、各タスクの目標日と「試験に間に合うか」を自動計算。週単位で日別の勉強時間を上書きできる
- **実績** — 見積もりと実績（実際にかかった時間）の差を確認
- **計画のインポート／エクスポート** — 計画・記録をJSONで書き出し・取り込み

## 使い方

初回は空の状態で開く。「プロジェクトを作る」から試験・教材・タスクを登録するか、`study-plan-import` 形式の計画JSONを「その他 → 計画を取り込む」で取り込む。

データは端末のブラウザにのみ保存される。機種変更やバックアップには「その他 → フルバックアップを書き出す」を使う。

## 技術

- 依存ライブラリなし・ビルド不要の単一 `index.html`
- 永続化: `localStorage`
- 配信: GitHub Pages / PWA（`manifest.json` + `sw.js`）
